# 二、安全区设置（必做，避免内容遮挡）

Figma 操作：选中框架 → 右侧「布局」面板 → 开启「安全区域」，核心内容（单词、按钮、图表）必须放在安全区内，具体参数如下：

|安全区位置|尺寸参数|适配说明|
|---|---|---|
|顶部安全区（含灵动岛）|≥59pt|导航栏需完全置于此区域，避免被灵动岛、状态栏遮挡|
|底部安全区（Home指示条）|≥34pt|Tabbar 放在此区域上方，不重叠，保证点击无遮挡|
|左右安全区|各16pt|与布局网格边距一致，单词卡片、列表左右贴边对齐|
|安全区内可用尺寸|358×751 pt|核心内容（单词卡片、统计图表）需在此范围内设计|

# 五、Figma 专属操作步骤（3分钟搞定）

1. 新建框架：打开 Figma → 新建文件 → 左侧「框架」工具 → 右侧属性面板输入 390×844pt，框架类型选 iOS，命名「背单词APP-iPhone15」，背景色设为 #F9F7F5。

2. 开启网格与安全区：选中框架 → 右侧「布局」→ 开启「安全区域」；添加列网格（12列、间距16pt、边距16pt）和基线网格（4pt），网格颜色设为 #E5E5E5（仅设计参考）。

3. 新建样式库：左侧「样式」面板 → 分别新建「颜色样式、文字样式、角半径样式」，按上述规范录入参数，命名清晰（如「颜色-主色-淡蓝」「文字-核心单词-28pt」）。

4. 搭建组件库：新建页面「组件库」→ 按上述组件尺寸绘制导航栏、Tabbar、单词卡片等 → 右键「创建组件」→ 归类到对应文件夹，后续直接拖拽复用。

5. 设置导出规范：选中组件/页面 → 右侧「导出」→ 格式选 PNG/SVG，缩放选 @1x/@2x/@3x，命名规范：「页面名_组件名_尺寸」（如「首页_单词卡片_358x180」）。

# 六、补充适配与协作建议

- 夜间模式：右键框架 → 创建变体 → 命名「日间/夜间」，替换对应颜色样式，一键切换预览。

- 大屏适配：复制主框架，修改尺寸为 iPhone 15 Pro Max（430×932pt），因采用网格+弹性间距，仅需微调卡片位置。

- 图层组织：按「页面→模块→元素」分层（如「首页→模块_AI学习卡→图片_AI搭子」），命名清晰，方便协作修改。

- 交互动效：右侧「原型」面板 → 添加页面跳转（淡入淡出0.2s）、卡片滑动、按钮缩放等动效，直接预览体验。
