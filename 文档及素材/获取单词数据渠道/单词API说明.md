**Free Dictionary API**

- 完全免费开源，不用密钥，直接调用就行。它能返回单词的发音、释义这些信息，不过单词分类得你自己做，你可以先把小学到大学还有 CET4、CET6 的单词表整理好，再用这个 API 去查每个单词的详情，这样就能实现分类背单词的功能



**skywind3000/ECDICT**

- 有 77 万多条，带音标和例句，你可以把这些词库按小学到大学的阶段整理一下，再对接 Free Dictionary API 获取单词详情。



**better-edu/dict**

- 里面有人教版小学到高中的分册词汇，比如三年级上册 64 个、六年级下册 108 个，初中七年级上册 392 个，都是按教材单元整理的，你可以直接下载用，省得自己分年级
