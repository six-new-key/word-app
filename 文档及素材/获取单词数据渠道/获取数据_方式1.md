# 免费获取单词数据方式



## 方式1

**KyleBing/english-vocabulary**

**中英文对照，英语词汇 txt json**

该库包含两个版本的词库
1. 乱序 txt 版本，只包含词条和注释，格式： `单词\t释义`
2. 顺序 json 版本，包含词条、多条注释、多条词组及注释

两个版本的词条数量是一样的

|      | 类别 | 单词数量 |
| ---- | ---- | -------: |
| 1    | 初中 |     3223 |
| 2    | 高中 |     6008 |
| 3    | 四级 |     7508 |
| 4    | 六级 |     5651 |
| 5    | 考研 |     9602 |
| 6    | 托福 |    13477 |
| 7    | SAT  |     8887 |
