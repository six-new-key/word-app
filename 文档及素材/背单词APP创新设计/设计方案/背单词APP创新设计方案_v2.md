# 背单词APP精简设计方案（功能模块+页面UI）

# 一、功能模块设计

## （一）基础核心功能（必开发）

1. 词汇管理：多维度分类（目标/难度/场景/记忆状态）、用户自定义分类标签

2. 自定义词库：支持手动/拍照/文本导入，AI自动去重、补充单词信息（音标/例句），支持备份、分享

3. 记忆学习：单词卡片（发音/词性/词义/例句）、4种基础测试模式（拼写/词义匹配/发音判断/例句填空）

4. 复习提醒：自定义提醒时间/频率，基于艾宾浩斯曲线推送易忘词，柔和弹窗提醒

5. 进度追踪：记录每日/每周/每月学习数据（新学/复习/正确率/遗忘率），动态统计图表展示

6. 基础体验：语音发音（英/美双音+语速调节）、夜间模式（手动+AI自动切换，护眼暖色调）

7. 社交互动：学习打卡海报/进度报告一键分享，好友互动（点赞/评论）、小型学习小组（互助复习）

## （二）AI创新功能（差异化，可落地）

1. AI个性化学习：动态规划记忆路径（贴合用户遗忘率）、智能推荐学习内容（适配学习目标）

2. AI单词辅助：词根词缀智能拆解（贴合用户记忆偏好）、场景化例句生成（适配用户学习场景）

3. AI发音纠错：跟读对比、逐音节纠正，生成个性化发音改进建议

4. AI错题管理：错题原因溯源、针对性复习计划生成，同类词拓展巩固

5. AI轻社交辅助：个性化打卡海报/学习报告生成、学习搭子（拟人化鼓励+简单单词互动）

# 二、页面UI设计

## 核心设计基调：柔和低饱和配色（浅米白/淡蓝/浅绿为主）、无衬线字体、充足留白、卡片式布局（圆角），护眼易操作

## 底部Tabbar（4个，最舒适布局，固定）

1. 首页：核心学习入口

2. 学习：单词学习+测试入口

3. 词库：词汇管理+自定义词库入口

4. 我的：进度统计+个人设置入口

## 各页面具体UI设计（只写核心布局/内容）

### 1. 首页（Tab1）

- 顶部：导航栏（APP名称+设置图标+夜间模式切换图标）

- 中间：今日学习卡片（目标+进度+AI学习搭子形象）、快捷功能（每日复习/记忆测试/AI对话，2x2卡片）、7天进度预览（简易折线图）

- 底部：固定Tabbar

### 2. 学习页（Tab2）

- 顶部：导航栏（返回+学习模式切换+进度条）

- 中间：单词展示（大号加粗+音标，点击发音）、核心词义/词性、AI辅助区（可折叠，词根拆解/场景例句）、互动区（左右滑动标记记忆状态，底部上/下单词按钮）

- 底部：固定Tabbar

### 3. 词库页（Tab3）

- 顶部：导航栏（返回+搜索栏+导入按钮）

- 中间：分类导航（横向滚动，推荐/自定义/目标/场景分类）、词库列表（卡片式，显示词库名称/单词量/难度）

- 底部：固定Tabbar

### 4. 我的页（Tab4）

- 顶部：个人信息（头像+昵称+学习等级）

- 中间：学习统计卡片（动态图表+核心数据）、功能设置（学习计划/提醒/AI设置/词库管理/帮助中心，卡片布局）、社交入口（学习小组/好友）

- 底部：固定Tabbar+版本信息/反馈按钮

### 5. 辅助页面（关联核心页面，无额外Tab）

- 测试页：从学习页/首页快捷入口进入，布局（题目+选项+提交按钮），结束后显示简易错题报告

- 统计页：从我的页进入，核心数据+动态图表（折线图/环形图），AI数据解读（简洁文字）

- 设置页：从我的页进入，分模块（学习设置/提醒设置/AI设置/夜间模式设置），简洁开关/输入框
